# PingARLT Font Files

This directory should contain the following PingARLT font files:

## Required Files:
- PingARLT-Thin.woff2
- PingARLT-Thin.woff
- PingARLT-Light.woff2
- PingARLT-Light.woff
- PingARLT-Regular.woff2
- PingARLT-Regular.woff
- PingARLT-Medium.woff2
- PingARLT-Medium.woff
- PingARLT-SemiBold.woff2
- PingARLT-SemiBold.woff
- PingARLT-Bold.woff2
- PingARLT-Bold.woff
- PingARLT-ExtraBold.woff2
- PingARLT-ExtraBold.woff
- PingARLT-Black.woff2
- PingARLT-Black.woff
- PingARLT-Heavy.woff2
- PingARLT-Heavy.woff

Please add these font files to this directory for the PingARLT font to work properly on the website.
